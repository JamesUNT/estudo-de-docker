⦁	Comando da aula pra levantar o banco:
docker run -it --rm -e POSTGRES_USER="root" --network=pg-network --name=pgdatabase -e POSTGRES_PASSWORD="root" -e POSTGRES_DB="ny_taxi" -v ny_taxi_postgres_data:/var/lib/postgresql -p 5432:5432 postgres:18

⦁	Comando para rodar o ingest_data.py:
docker run -it --rm --network=pg-network taxi_ingest:v001 --pg-user=root --pg-password=root --pg-host=pgdatabase --pg-port=5432 --pg-db=ny_taxi --target-table=yellow_taxi_trips --year=2021 --month=1 --chunksize=100000

⦁	Comando para abrir o postgress admin no docker:
docker run -it -e PGADMIN_DEFAULT_EMAIL="admin@admin.com" -e PGADMIN_DEFAULT_PASSWORD="root" -v pgadmin_data:/var/lib/pgadmin -p 8085:80 --network=pg-network --name pgadmin dpage/pgadmin4
